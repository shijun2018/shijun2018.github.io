Maven安装

Maven

要安装 Apache Maven 在Windows 系统上, 需要下载 Maven 的 zip 文件，并将其解压到你想安装的目录，并配置 Windows 环境变量。

安装Apache Maven

Apache Maven的安装是一个简单的过程，它可以提取档案并将mvn命令的bin文件夹添加到PATH中。
详细步骤如下：

  • 确保设置了JAVA_HOME环境变量并指向您的JDK安装
  • 在任何目录中提取分发存档

或

  • 将创建的目录apache-maven-3.6.2的bin目录添加到PATH环境变量中
  • 在新的shell中用mvn -v确认。结果应类似于

Windows提示

  • 检查环境变量值，例如
  • 添加到PATH：打开系统属性（WinKey +暂停），选择“高级”选项卡和“环境变量”按钮，然后添加或选择PATH变量，将解压后的发行版的bin目录添加到用户PATH环境变量中在用户变量中的值为C：\ Program Files \ apache-maven-3.6.2 \ bin。可以使用同一对话框将JAVA_HOME设置为JDK的位置，例如C：\ Program Files \ Java \ jdk1.7.0_51
  • 打开一个新的命令提示符（Winkey + R，然后键入cmd）并运行mvn -v验证安装。
